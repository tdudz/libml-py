# libml-py
A machine learning library in python, written from scratch.

Requirements:
* `numpy`: for handling matrices/vectors

Features:
* **Descent Methods:**
  * Batch Gradient Descent
  * Stochastic Gradient Descent
  * Minibatch Gradient Descent
  * Newton's Method
* **Regression:**
  * Linear Regression 
* **Classification:**
  * Logistic Regression
